# 🎓 Student API

## 📌 Description
API simple en PHP pour ajouter un étudiant dans une base de données MySQL.

## ⚙️ Technologies
- PHP (XAMPP)
- MySQL
- Postman

## 🚀 Utilisation

**URL :**
http://localhost/studentapi/api/addStudent.php

**Méthode :** POST  

**Body (x-www-form-urlencoded) :**
- nom
- prenom
- ville
- sexe

## ✅ Résultat
Ajout d’un étudiant + réponse JSON
